Reinstate the WP Editor on the "page for posts" page.

== Description ==

This plugin reinstates the content/wp-editor field that was removed from the "page for posts" page in the 4.2.2 update.

== Installation ==

1. Upload `add-editor-to-page-for-posts.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're done.
